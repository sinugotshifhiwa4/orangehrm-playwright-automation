# OrangeHRM Playwright Automation Framework

A **Playwright + TypeScript** end-to-end UI automation framework for the [OrangeHRM Demo](https://opensource-demo.orangehrmlive.com) application.

Built with a **Page Object Model (POM)** architecture, custom fixtures, reusable utilities, and a CI/CD-ready structure for scalable and maintainable test automation.

---

## Prerequisites

- [Node.js](https://nodejs.org) v18+
- npm v9+

---

## Getting Started

Install dependencies:

```bash
npm install
```

Install Playwright browsers:

```bash
npx playwright install
```
